# Public Boundary

This repository is meant for judges, sponsors, and people reviewing the HackRome submission. It is not the Netify product source code.

## Included

- Devpost submission copy.
- Public project explanation.
- Explainer PDF and preview image.
- API contract, OpenAPI schema, and small client examples.
- High-level architecture and hackathon progress notes.

## Excluded

- Proprietary PDF-to-video pipeline source.
- Generated Remotion scene code.
- Render repair and validation logic.
- Prompt chains and model orchestration details.
- Private web app source code.
- Supabase service-role keys, ElevenLabs keys, LLM keys, and environment files.
- Private input PDFs, generated intermediate assets, and local render outputs.

## Why

Netify's core value is the generation system: document understanding, lesson planning, animated code generation, voice timing, validation, repair, and rendering. This public repository shows what was built and how the product works without making the implementation copyable.
